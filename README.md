# Affordmed Full Stack Assessment

This repository contains my submission for the Affordmed Campus Hiring Evaluation.

## Project Structure

* `logging_middleware` - Custom logging utility
* `notification_app_be` - Backend implementation
* `notification_app_fe` - Frontend application
* `notification_system_design.md` - System design and architecture documentation

## Features

* Notification management system
* Priority inbox implementation
* Logging middleware integration
* Responsive frontend UI
* Notification filtering and pagination

## Tech Stack

* TypeScript
* React
* Node.js
* Material UI

Developed as part of the Affordmed Full Stack Assessment.
